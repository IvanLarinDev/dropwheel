# Mockups - open-animation-settings

DESIGN stage for adding a user-selectable wheel open animation in Settings.

Options:

1. `01-compact-combo.html` - minimal change to the existing settings window; a combo box with a small inline preview.
2. `02-preview-first.html` - larger visual preview above the surrounding settings controls.
3. `03-motion-list.html` - radio-list selection with short descriptions and mini motion strips.
4. `04-system-plain.html` - conservative grouped settings dialog with plain Windows-style controls.

The implementation should keep the current compact WPF settings surface unless the approved mockup says otherwise.
