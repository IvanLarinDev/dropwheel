# tray-menu-icons — chosen variant

Four variants for tidying the tray menu are in `mockups.html`.

**Chosen: Variant 3 — icons + accent checkmarks + grouping.**

The menu keeps its length and item order but gains separators between the four
semantic groups (system integration, pauses, settings & config file, history)
and a monochrome Segoe MDL2 Assets glyph in front of every action item, tinted
with the palette's muted text color. Checkable toggles show the accent-colored
check glyph when on and their own muted topic glyph when off — the stock check
boxes are suppressed by a custom renderer, which is now used for light themes
too so both look consistent.
