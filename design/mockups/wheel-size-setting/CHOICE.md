# Wheel size — ползунок размера колеса

- Дата: 2026-07-15
- Одобрил: Ivan Larin
- Выбрано: один слайдер «Wheel size» в секции Appearance, диапазон 0.8–1.5, шаг 0.05.

## Что менялось

Добавлен пользовательский масштаб размера колеса. В настройках (Appearance) — слайдер
«Wheel size» рядом с прозрачностью орба и скоростью анимации, по образцу существующих
слайдеров (значение показывается как «1.00x»). При сохранении значение клампится через
`WheelLayout.ClampScale` и применяется в `ApplySettings`, где уже пересчитываются размер
фиксированного окна, `ScaleTransform` на слое колеса и зоны наведения орба (обвязка —
пакет wheel-scale-plumbing). Масштаб растит кольцо, тайлы, орб и подписи вместе, не трогая
базовую геометрию.

## Почему один слайдер, а не набор параметров

Отдельные ползунки для радиуса, тайла и орба легко привести в нелепое сочетание и требуют
параметризации хрупкой выверенной геометрии (офсеты тайла и спицы подобраны вручную, без
чистой формулы). Один общий масштаб через `ScaleTransform` проще, безопаснее и при 1.0
пиксель в пиксель совпадает с классическим колесом. Превью диапазона — `mockups.html`.
